- javascript embeded umumnya ditulis setelah elemen html ```<body>```:
```html
<script>
    var p = document.getElementById('product-grid')
</script>
```

- CSS embeded ditulis di antara tag ```<head>``` dan ```</head>```dengan menggunakan :

```html
<style>
    body{
        background-color: #f7f9fc;
        color: #333;
        line-height: 1.6;
    }
</style>
```
